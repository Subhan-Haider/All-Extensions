## üöÄ Getting Started

To test any of these extensions locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Subhan-Haider/All-Extensions.git
    cd All-Extensions
    ```

2.  **Load into Chrome/Edge**:
    *   Open your browser and navigate to `chrome://extensions` (or `edge://extensions`).
    *   Enable **Developer Mode** (usually a toggle in the top right).
    *   Click **Load Unpacked**.
    *   Select the specific folder of the extension you want to use (e.g., `./AdShield-Pro`).
