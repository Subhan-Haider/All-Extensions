# Extension Icons

This folder should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create simple icons using any image editor:

1. Create a square image (e.g., 512x512)
2. Design a simple icon (e.g., globe, download, or website symbol)
3. Export as PNG at different sizes:
   - 16x16
   - 48x48
   - 128x128

## Temporary Solution

For testing, you can:
1. Use any PNG images renamed to the required names
2. Or create simple colored squares as placeholders
3. The extension will work without icons, but Chrome may show warnings
